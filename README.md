# Solution 8 Project Toolkit

This repository contains the **Solution 8 Project Toolkit** - a comprehensive collection of activities, workshops, tools, and materials for complex system design and design thinking projects.

Solution 8 is a technology agency with deep expertise in complex system design and design thinking.

## About This Site

This documentation site is built using [Jekyll] with the [Just the Docs] theme and is published via [GitHub Pages].

The toolkit includes methodologies and frameworks for:

- **Team Collaboration**: Building trust, establishing communication norms, and aligning team members
- **User Research**: Understanding user needs, conducting discovery, and synthesizing insights
- **Product Strategy**: Defining vision, prioritizing solutions, and planning roadmaps
- **Design & Development**: Story mapping, wireframing, and iterative development practices

## Building and Previewing Locally

To build and preview the site locally:

1. Install [Jekyll] and [Bundler]
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Open your browser to `localhost:4000`

## Publishing

This site is automatically built and published to GitHub Pages using GitHub Actions whenever changes are pushed to the `main` branch.
